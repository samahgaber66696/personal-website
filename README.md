# personal-website
This is my solution to a Frontend Mentor challenge. The goal of this project was to build a responsive and accessible web page based on the provided design while following modern front-end development practices.

## Overview

This project was created as part of a challenge from Frontend Mentor to improve my HTML, CSS, and responsive design skills. The challenge helped me practice building layouts, organizing styles, and writing maintainable code.

## Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Sass (SCSS)

## Features

- Responsive layout for different screen sizes
- Modern CSS layout techniques using Flexbox and Grid
- Organized and maintainable Sass structure
- Semantic HTML markup
- Clean and accessible user interface

## What I Learned

While working on this project, I strengthened my understanding of:

- Creating responsive layouts with Flexbox and CSS Grid
- Structuring styles efficiently using Sass
- Writing semantic and accessible HTML
- Following a design specification and translating it into code
- Improving code organization and maintainability

## Continued Development

In future projects, I would like to:

- Improve accessibility even further
- Enhance animations and interactions
- Explore more advanced Sass features
- Optimize performance and code structure
